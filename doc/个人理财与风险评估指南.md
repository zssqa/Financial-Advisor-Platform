# 个人理财规划与风险评估

## 一、理财规划的完整框架

### 1.1 七步理财法

```
第一步：梳理财务现状        → 盘点资产（存款、房产、基金等）与负债（房贷、车贷、信用卡欠款）
第二步：设定理财目标        → 分短期(1年内)、中期(1-3年)、长期(3年以上)列出可量化目标
第三步：建立应急准备金       → 预留3-6个月生活费于货币基金或活期存款，专款专用
第四步：配置保障体系         → 重疾险+医疗险+意外险+定期寿险，保费控制在年收入5%-10%
第五步：投资增值             → 按风险承受能力选择合适的投资组合
第六步：税务与债务管理       → 合理规划专项附加扣除，优化高息债务结构
第七步：定期复盘              → 每半年或每年检查一次组合表现，调整偏离
```

### 1.2 收支分析模板

**每月净现金流 = 税后收入 - 必要支出 - 储蓄/投资金额**

| 项目 | 占比建议 | 说明 |
|------|---------|------|
| 住房支出（房租/房贷） | ≤ 30% | 超过此比例将严重影响生活质量 |
| 日常消费（餐饮/交通/日用） | 20%-25% | 视城市等级和个人习惯浮动 |
| 保险保障 | 5%-10% | 建议先保大病和意外，再考虑年金/理财险 |
| 教育培训 | 5%左右 | 提升自身生产力是最有效的投资 |
| 娱乐休闲 | 5%-10% | 保持合理享受，但需有预算上限 |
| 储蓄与投资 | 至少10%，理想20%+ | 越早开始复利效应越显著 |

## 二、金融需求综合评估模型

### 2.1 评估问卷说明

本系统采用标准的 **R1-R5 五档风险评估体系**，共覆盖以下维度：

#### 维度一：基本情况

| 编号 | 问题 | 选项及分值 |
|------|------|-----------|
| Q1 | 您的年龄段？ | 25岁以下(1) / 25-35岁(2) / 35-50岁(3) / 50岁以上(4) |
| Q2 | 您的教育程度？ | 高中及以下(1) / 大专(2) / 本科(3) / 硕士及以上(4) |
| Q3 | 您的婚姻状况？ | 单身(2) / 已婚无子女(3) / 已婚有子女(2) |
| Q4 | 您目前的家庭负担情况？ | 一人养家(1) / 两人分担(2) / 有老人赡养(1) |

#### 维度二：财务状况

| 编号 | 问题 | 选项及分值 |
|------|------|-----------|
| Q5 | 您的月收入水平？ | 8千以下(1) / 8千-1.5万(2) / 1.5-3万(3) / 3万以上(4) |
| Q6 | 您的金融资产规模？ | 10万以下(1) / 10-50万(2) / 50-200万(3) / 200万以上(4) |
| Q7 | 是否有固定投资和理财经验？ | 无任何经验(0) / 1年以下(1) / 1-3年(2) / 3年以上(3) |
| Q8 | 投资资产占全部资产的比重？ | 50%以下(1) / 50%-75%(2) / 75%以上(3) |

#### 维度三：风险偏好

| 编号 | 问题 | 选项及分值 |
|------|------|-----------|
| Q9 | 最期望的年投资收益率？ | 3%以下(1) / 3-8%(2) / 8-15%(3) / 15%以上(4) |
| Q10 | 能否接受的最大亏损幅度？ | 不接受(0) / 5%以内(1) / 5-15%(2) / 15-30%(3) / 30%以上(4) |
| Q11 | 如投资下跌，您将如何处理？ | 立即清仓(1) / 持有不动(2) / 逢低补仓(3) |
| Q12 | 您的投资期限一般为多久？ | 小于1年(1) / 1-3年(2) / 3-5年(3) / 5年以上(4) |

#### 维度四：心理特征

| 编号 | 问题 | 选项及分值 |
|------|------|-----------|
| Q13 | 当周围人赚钱时您是否焦虑？ | 经常焦虑(1) / 偶尔焦虑(3) / 基本不焦虑(5) |
| Q14 | 面对市场波动，您的第一反应？ | 恐慌(1) / 观望(3) / 寻找机会买入(5) |
| Q15 | 您认为最好的投资机会是？ | 银行存款(1) / 债券(2) / 指数基金(3) / 股票(5) |

**评分标准：**
- 总分范围：约 0-75 分
- **R1（极度保守）**：15分以下 —— 保本优先，只接受低风险产品
- **R2（偏保守）**：15-30分 —— 轻微风险可接受，以稳健型产品为主
- **R3（稳健平衡）**：30-50分 —— 能平衡收益和风险，可采用股债混合策略
- **R4（偏激进）**：50-65分 —— 追求较高回报，可承受较大波动
- **R5（极度激进）**：65分以上 —— 追求最大化收益，可接受大幅回撤

### 2.2 结果应用

不同风险等级匹配的投资工具：

| 风险等级 | 推荐工具 | 预期年化 | 典型最大回撤 |
|---------|---------|---------|------------|
| R1 | 货币基金、国债逆回购、大额存单 | 1.5%-2.5% | < 0.1% |
| R2 | 短债基金、同业存单指数基金、银行R2理财 | 2.5%-4% | < 1% |
| R3 | 纯债基金、固收+、宽基指数定投 | 4%-8% | 5%-15% |
| R4 | 行业基金、QDII、可转债、少量个股 | 8%-15% | 15%-30% |
| R5 | 股票、行业主题基金、杠杆ETF | 10%+ | 30%-50% |

## 三、常见理财产品深度对比

### 3.1 七大主流理财产品一览

| 产品类型 | 门槛 | 流动性 | 风险等级 | 历史年化 | 适合人群 |
|---------|------|--------|---------|---------|---------|
| 活期存款 | 1元 | T+0 | 极低 | ~0.3% | 所有人群 |
| 货币基金 | 1元 | T+0/T+1 | 极低 | ~1.5-2% | 闲置资金管理 |
| 银行理财(R1/R2) | 1万起 | 封闭期不等 | 低-中低 | 2.5-4% | 稳健投资者 |
| 纯债基金 | 10元起 | T+1~T+3 | 低 | 3-5% | 有一定耐心者 |
| 固收+基金 | 1元起 | T+1~T+3 | 中 | 4-7% | 能承受小幅波动者 |
| 指数基金 | 10元起 | T+1 | 中高 | 6-12% | 中长期投资者 |
| 主动权益基金 | 10元起 | T+1~T+3 | 高 | 8-15% | 高风险承受力者 |

### 3.2 保险产品的功能定位

保险不是用来赚大钱的，核心功能是**转移重大风险**。优先级建议：

```
第一优先 → 百万医疗险（几百元/年，保额可达数百万，覆盖住院费用）
第二优先 → 重疾险（一次性赔付，弥补收入损失，保额建议30-50万）
第三优先 → 意外险（百元左右/年，覆盖意外身故和伤残）
第四优先 → 定期寿险（如有家庭责任，保额应覆盖房贷+子女教育费用）
最后考虑 → 年金险/增额终身寿（用于养老规划和财富传承，本质是长期储蓄）
```

## 四、储蓄目标规划

### 4.1 复利计算器关键参数

**复利公式：** $FV = PV \times (1 + r)^n$

其中：
- $FV$ = 终值（未来值）
- $PV$ = 现值（初始本金）
- $r$ = 年利率
- $n$ = 年数

**实例：** 月存5000元，年化收益5%，连续存10年：
$$总额 = 5000 \times 12 \times \frac{(1.05^{10} - 1)}{0.05} \approx 680,405 \text{元}$$
其中本金60万，利息约8万。若持续20年则达约187万元，利息部分远超本金——这就是长期投资的威力。

### 4.2 常见储蓄目标速算表

| 目标 | 所需金额 | 时间 | 月均需投入（假设年化5%） |
|------|---------|------|-----------------------|
| 应急储备金 | 3-6个月开支 | 立即 | 根据实际生活支出计算 |
| 购房首付（二线城市） | 60-100万 | 5年 | 约1-1.5万元 |
| 子女大学教育金 | 30-50万 | 15-18年 | 约800-1200元 |
| 退休养老金（补充） | 200-500万 | 20-30年 | 根据已有积累测算 |

### 4.3 信用卡分期还款成本计算

**隐藏陷阱：信用卡分期付款名义利率看似很低（年费率约0.6%），但实际年化利率可能高达1.3%-1.5%。**

原因：分期手续费按**总本金**每月收取，而非按剩余本金收取。例如：
- 分12期，每期费率0.6%，月供 = 本金/12 + 本金×0.6%
- 但到第6期时你已还了一半本金，却仍然在付全额本金的手续费
- **真实年化利率 ≈ 月费率 × 12 × 1.8（近似公式）**

| 期数 | 表面费率(%) | 实际APR(%) | 差异倍数 |
|------|-----------|-----------|---------|
| 6期 | 0.60 | 13.0 | 2.17x |
| 12期 | 0.60 | 13.0 | 2.17x |
| 24期 | 0.55 | 12.0 | 2.18x |

## 五、汇率与跨境资产配置

### 5.1 人民币兑美元汇率基础

当前人民币汇率形成机制参考"一篮子货币"，受以下因素影响：

1. **中美利差**：美联储加息→美元走强→人民币承压
2. **经济基本面**：中国GDP增速放缓→外资流出→人民币贬值压力
3. **政策干预**：央行通过逆周期因子、外汇存款准备金率等工具调节
4. **国际贸易顺差**：持续顺差推升人民币需求

### 5.2 汇率换算实用方法

```
汇率换算 = 原币金额 × 实时汇率

示例：1万美元 → 人民币
假设汇率 7.25：10,000 × 7.25 = 72,500 元
```

本项目 `ExchangeRateTool` 支持主要货币对（USD/EUR/JPY/HKD等）的实时查询和换算，数据来源于外管局公开报价和市场中间价。

## 六、个人所得税计算要点

### 6.1 综合所得计税方式

自2019年起，中国实行 **累计预扣法** 计算工资薪金个税：

**年度应纳税所得额 = 年收入 - 60000(起征点) - 三险一金 - 专项附加扣除**

税率采用七级累进：

| 级数 | 应纳税所得额区间 | 税率 | 速算扣除数 |
|------|----------------|------|-----------|
| 1 | 不超过36,000元 | 3% | 0 |
| 2 | 36,000-144,000元 | 10% | 2,520 |
| 3 | 144,000-300,000元 | 20% | 16,920 |
| 4 | 300,000-420,000元 | 25% | 31,920 |
| 5 | 420,000-660,000元 | 30% | 52,920 |
| 6 | 660,000-960,000元 | 35% | 85,920 |
| 7 | 超过960,000元 | 45% | 181,920 |

| 项目 | 扣除标准 | 备注 |
