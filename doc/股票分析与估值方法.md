# 股票分析与估值方法

## 一、基本面分析框架

### 1.1 自上而下三层分析法

```
宏观分析 → 判断经济周期位置和政策方向，决定大类资产配置比例（股/债）
          ↓
行业分析 → 寻找景气向上或政策扶持的行业，缩小选股范围
          ↓
个股分析 → 深入财报和估值，挑选优质公司并确定买卖时机
```

### 1.2 宏观经济关键指标

| 指标 | 含义 | 对市场的影响 |
|------|------|------------|
| GDP增速 | 国民经济总体增速 | 增速回升→利好股市，反之承压 |
| CPI/PPI | 消费者/生产者价格指数 | CPI过高→加息预期→利空；PPI上行→企业利润改善→利好 |
| PMI（采购经理指数） | 制造业景气度先行指标 | >50扩张，<50收缩；PMI拐点常领先市场3-6个月 |
| M2增速 | 货币供应量 | 流动性宽松→利好资产价格 |
|社融/信贷增量 | 实体融资需求 | 反映经济活跃度 |
| SHIBOR/LPR | 银行间利率/贷款基准利率 | 降息周期利好成长股估值修复 |

## 二、财务报表三大表解读

### 2.1 资产负债表——公司的"底子"

资产负债表反映企业在某一**特定时点**的财务状况。核心公式：

$$资产 = 负债 + 所有者权益$$

#### 重点科目解析

| 科目 | 关注要点 | 健康标准 |
|------|---------|---------|
| 货币资金 | 是否够覆盖短期债务 | 流动比率(流动资产/流动负债) > 1.5 |
| 应收账款 | 回收风险 | 应收占总资产比例不宜超过20% |
| 存货 | 跌价风险 | 存货周转率应与同行对比 |
| 固定资产 | 重资产还是轻资产 | 互联网/软件公司通常较低 |
| 商誉 | 并购溢价形成的无形资产 | 商誉/净资产 < 10%为安全线 |
| 有息负债 | 需要付利息的债务 | 资产负债率 < 60%较稳健 |
| 未分配利润 | 历年累积留存收益 | 持续增长为佳 |

### 2.2 利润表——公司的"面子"

利润表反映一个会计期间的经营成果：

**营收 → 毛利 → 营业利润 → 净利润**

| 利润率 | 公式 | 评价标准 |
|-------|------|---------|
| 毛利率 | (营业收入 - 营业成本) / 营业收入 | 越高越好，一般>30%有竞争力 |
| 净利率 | 净利润 / 营业收入 | ROE的组成部分，需结合杜邦分析看 |
| 营业利润率 | 营业利润 / 营业收入 | 剔除非经常性损益的真实经营能力 |

**盈利质量检查清单：**
- [ ] 经营性现金流净额是否持续为正且接近或大于净利润？
- [ ] 扣非净利润与净利润差距是否过大？（可能有大量一次性收益掩盖主业不佳）
- [ ] 毛利率是否逐年下降？（可能竞争力在衰退）

### 2.3 现金流量表——公司的"日子"

现金流量表是最难做假的一张报表，重点关注四类活动现金流：

| 现金流类型 | 理想状态 | 警示信号 |
|-----------|---------|---------|
| 经营活动现金流 | 持续为正且增长 | 连续多个季度为负或波动剧烈 |
| 投资活动现金流 | 一般为负（在投入扩张） | 大量变卖资产维持运营 |
| 筹资活动现金流 | 视阶段而定 | 频繁增发稀释股东权益 |
| **期末自由现金流** | 为正（经营现金流-资本开支） | 长期为负说明公司在"烧钱" |

**自由现金流（FCF）是估值最重要的输入变量：**
$$FCF = 经营活动现金流 - 资本性支出$$
$$企业价值 ≈ \sum_{t=1}^{n}\frac{FCF_t}{(1+r)^t} + 终值$$

## 三、相对估值法

### 3.1 PE（市盈率）估值

$$PE = \frac{股价}{每股收益(EPS)} = \frac{总市值}{净利润}$$

| PE区间 | 含义 | 适用场景 |
|--------|------|---------|
| PE < 10 | 低估区域（视行业而定） | 成熟行业（银行、公用事业）可用 |
| PE 10-20 | 合理区间 | 多数行业常态 |
| PE 20-40 | 偏高但可接受 | 高成长性行业可容忍较高PE |
| PE > 40 | 高风险区域 | 需验证高速增长是否可持续 |

**使用注意：**
- PE适合盈利稳定的公司，亏损企业PE为负无参考意义
- 周期股（钢铁、煤炭、化工）往往在PE最低时买入，PE最高时卖出——因为盈利随周期波动
- 建议使用**滚动市盈率（TTM）**而非静态PE

### 3.2 PB（市净率）估值

$$PB = \frac{总市值}{净资产} = \frac{股价}{每股净资产}$$

| PB区间 | 适用资产类型 |
|--------|-------------|
| PB < 1 | 破净，常见于银行、保险等金融行业 |
| PB 1-2 | 正常偏低，适用于重资产制造业 |
| PB 2-5 | 适中，常见于消费/医药等行业龙头 |
| PB > 5 | 偏贵或轻资产高ROE公司（如互联网平台） |

### 3.3 PEG（市盈率相对盈利增长率）

$$PEG = \frac{PE}{净利润复合增长率(\%)}$$

- PEG < 1：低估（如PE=30，增长率30%，PEG=1.0）
- PEG = 1：合理
- PEG > 2：明显高估

**适用条件：** 仅适用于有明确成长逻辑的公司，不适合零增长或负增长企业。

### 3.4 股息率（Dividend Yield）

$$股息率 = \frac{每股分红}{股价} \times 100\%$$

| 股息率 | 评价 |
|-------|------|
| > 5% | 高分红，适合防御型配置 |
| 3%-5% | 合理 |
| < 2% | 低分红，可能公司将利润用于再投资 |

## 四、绝对估值法 —— DCF折现模型

### 4.1 原理

将公司未来所有自由现金流折现到当前时点，求出内在价值：

$$V_0 = \sum_{t=1}^{n}\frac{FCF_t}{(1+WACC)^t} + \frac{TV}{(1+WACC)^n}$$

其中：
- $FCF_t$ = 第t年的自由现金流
- $WACC$ = 加权平均资本成本（股权成本和债务成本的加权平均）
- $TV$ = 终值（通常用永续增长法：$TV = \frac{FCF_n \times (1+g)}{WACC-g}$）

### 4.2 WACC计算公式

$$WACC = \frac{E}{E+D} \times r_e + \frac{D}{E+D} \times r_d \times (1-T_c)$$

- $E$ = 股权市值
- $D$ = 债务市值
- $r_e$ = 股权成本（常用CAPM模型计算：$r_e = r_f + \beta \times (r_m - r_f)$）
- $r_d$ = 债务成本（通常为借款利率）
- $T_c$ = 企业所得税率

### 4.3 DCF模型的局限性

1. **高度依赖假设**：增长率、折现率的微小变化可能导致结果差异巨大
2. **不适合早期企业**：现金流不稳定的初创公司难以可靠预测
3. **终端值占比过高**：往往终值占企业总价值的50%以上，不确定性大

**建议：** DCF主要用于交叉验证，不应单独作为决策依据。

## 五、K线技术分析入门

### 5.1 K线基础形态

| 形态名称 | 图形特征 | 含义 |
|---------|---------|------|
| 阳线 | 收盘价 > 开盘价 | 买方力量占优 |
| 阴线 | 收盘价 < 开盘价 | 卖方力量占优 |
| 十字星 | 开盘≈收盘，上下影线较长 | 多空博弈激烈，转折信号 |
| 锤子线 | 下影线长，上影线极短或无 | 底部反转信号 |
| 倒锤头 | 上影线长，实体小 | 上方抛压沉重 |
| 吞没形态 | 后一根K线完全包住前一根 | 趋势反转强烈信号 |

### 5.2 常用技术指标

| 指标 | 参数 | 含义 | 用法 |
|------|------|------|------|
| MA（移动平均线） | 5日/10日/20日/60日/120日/250日 | 平滑价格走势 | 多头排列（短均线在上）看涨，空头排列看跌 |
| MACD | 快线12、慢线26、信号线9 | 趋势跟踪动量指标 | DIF上穿DEA金叉买入，下穿死叉卖出 |
| KDJ | 参数9,3,3 | 随机震荡指标 | K>D金叉买入，K<D死叉卖出，超买>80，超卖<20 |
| RSI | 参数14 | 相对强弱指标 | RSI>80超买区，RSI<20超卖区 |
| 布林带(BOLL) | 中轨20日MA，上下轨±2σ | 波动区间 | 触及上轨可能回落，触及下轨可能反弹 |

### 5.3 成交量分析原则

- **量价齐升** → 健康上涨趋势
- **缩量上涨** → 动能不足，警惕回调
- **放量下跌** → 恐慌出逃，短期可能继续下探
- **低位放量** → 可能是底部信号
- **高位放量滞涨** → 主力出货嫌疑

## 六、财务健康诊断工具

### 6.1 Altman Z-Score破产预警模型

适用于上市公司：

$$Z = 1.2X_1 + 1.4X_2 + 3.3X_3 + 0.6X_4 + 0.99X_5$$

| 变量 | 定义 |
|------|------|
| X1 | 营运资金 / 总资产 |
| X2 | 留存收益 / 总资产 |
| X3 | 息税前利润 / 总资产（ROA） |
| X4 | 股权市值 / 总负债 |
| X5 | 营业收入 / 总资产（资产周转率） |

| Z值区间 | 危险等级 |
|---------|---------|
| Z < 1.81 | 高风险（可能在2年内破产） |
| 1.81 ≤ Z ≤ 2.99 | 灰色地带 |
| Z > 2.99 | 安全区域 |

### 6.2 杜邦分析法——拆解ROE

$$ROE = 净利率 \times 总资产周转率 \times 权益乘数$$

- **净利率**：产品盈利能力（毛利率-费用率）
- **总资产周转率**：管理效率（收入/总资产）
- **权益乘数**：杠杆水平（总资产/权益）

通过杜邦分析可以识别公司是**靠什么赚钱的**：
- 茅台：高净利率驱动（产品力强）
- 沃尔玛：高周转率驱动（薄利多销）
- 银行：高杠杆驱动（放大规模效应）
