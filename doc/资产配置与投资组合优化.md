# 资产配置与投资组合优化

## 一、资产配置的核心逻辑

### 1.1 为什么需要资产配置？

投资界有一句名言："**投资收益的90%来自资产配置，而非个股选择或择时。**"——Brinson, Hood & Beebower (1986) 经典研究的结论至今仍被广泛引用。

配置的本质是：通过持有相关性低甚至负相关的资产类别，在不降低收益的前提下显著降低整体组合波动。

#### 经典案例对比

| 组合类型 | 预期年化 | 最大回撤 | 夏普比率 |
|---------|---------|---------|---------|
| 纯股票（100%沪深300） | ~8-10% | -40%至-50% | 约0.4-0.6 |
| 股债混合（60%股票+40%债券） | ~7-9% | -20%至-25% | 约0.8-1.0 |
| 全球多元（A股+港股+美股+黄金+债券） | ~6-8% | -15%至-20% | 约1.0-1.3 |

可见，多元化配置在**几乎不牺牲长期回报**的情况下大幅改善风险收益比。

## 二、马科维茨现代投资组合理论（MPT）

### 2.1 理论基础

哈里·马科维茨（Harry Markowitz, 1952）提出：**投资者不应关注单个资产的期望收益和风险，而应关注它在整个组合中所起的作用。**

核心公式：

$$E(R_p) = \sum_{i=1}^{n} w_i E(R_i)$$

$$\sigma_p^2 = \sum_{i=1}^{n}\sum_{j=1}^{n} w_i w_j \sigma_i \sigma_j \rho_{ij}$$

其中：
- $E(R_p)$：组合期望收益率
- $w_i$：资产 $i$ 的投资权重
- $\sigma_p$：组合标准差（总风险）
- $\rho_{ij}$：资产 $i$ 和 $j$ 的相关系数

### 2.2 有效前沿与最优组合

- **有效前沿（Efficient Frontier）**：在给定风险水平下能获得最高期望收益的所有组合的集合
- **切点组合（Tangency Portfolio）**：无风险利率与有效前沿相切的组合，即夏普比率最高的组合
- **资本市场线（CML）**：从无风险利率出发经过切点组合的直线，代表最优风险-收益路径

### 2.3 实际计算步骤

#### 第一步：收集数据
获取各资产的历史收益率序列（建议至少3年日频或周频数据）。

#### 第二步：计算统计量
- 各资产平均收益率 $E(R_i)$
- 各资产标准差 $\sigma_i$
- 资产间相关系数矩阵 $\rho$

#### 第三步：蒙特卡洛模拟生成随机组合
```
for i in 1 to 10000:
    weights = random_weights(n_assets)  # 约束 sum(weights)=1
    portfolio_return = dot(weights, mean_returns)
    portfolio_variance = t(weights) @ covariance_matrix @ weights
    portfolio_sharpe = (portfolio_return - rf) / sqrt(portfolio_variance)
```

#### 第四步：筛选最优解
- 取所有模拟组合中夏普比率最高的一个作为切点组合
- 输出各资产的最优配置比例

## 三、常见配置模型

### 3.1 经典策略

| 策略名称 | 配置方案 | 适用人群 | 特点 |
|---------|---------|---------|------|
| **股债平衡** | 股票:债券 = 60:40 或 50:50，每年再平衡一次 | 稳健型投资者 | 简单有效，纪律性强 |
| **永久组合** | 股票40%+债券55%+黄金5%+现金0%（或等份四等分） | 极度保守者 | 无需择时，穿越牛熊 |
| **核心-卫星** | 核心70%宽基指数+卫星30%行业/主题基金 | 有一定经验者 | 兼顾稳健与弹性 |
| **目标日期** | 随目标年份临近自动降低股票比例 | 养老金/教育金规划 | 生命周期动态调整 |
| **风险平价** | 按风险贡献等分而非按金额等分 | 专业投资者 | 更均衡的风险暴露 |

### 3.2 股债平衡策略详解（推荐初学者）

```
初始配置：60% 股票类 + 40% 债券类

再平衡规则：每季度检查一次
├─ 如果股票涨到占组合的70%，卖出股票买入债券 → 回到60:40
└─ 如果股票跌到占组合的50%，卖出债券买入股票 → 回到60:40

效果：自动实现"高抛低吸"，长期年化可比单纯持有提升1-2个百分点
```

## 四、个人化配置方案模板

### 4.1 风险评估结果应用

根据风险评估问卷（R1-R5）获得风险等级后，可参考以下配置区间：

| 风险等级 | 保守型(R1) | 略保守(R2) | 稳健型(R3) | 略激进(R4) | 激进型(R5) |
|---------|-----------|-----------|-----------|-----------|-----------|
| 股票/权益基金 | 0%-20% | 20%-40% | 40%-60% | 60%-80% | 80%-100% |
| 债券/固收 | 60%-80% | 40%-60% | 30%-50% | 15%-30% | 0%-15% |
| 现金/货币基金 | 10%-20% | 10%-15% | 5%-10% | 5% | 0%-5% |
| 另类(黄金/商品/QDII) | 0%-5% | 0%-5% | 5%-10% | 5%-10% | 5%-15% |

### 4.2 按理财目标定制

| 目标 | 资金需求期限 | 推荐配置建议 |
|------|------------|------------|
| 养老储备 | 20年以上 | 偏股型为核心，逐步过渡到固收为主 |
| 子女教育 | 10年左右 | 中期以固收+为底仓，后期逐步降风险敞口 |
| 购房首付 | 3年内 | 以短债基金和货币基金为主，保持流动性 |
| 日常增值 | 不限 | 可按股债平衡策略长期执行 |
| 应急准备金 | 随时可用 | 货币基金或活期存款，保证流动性第一 |

## 五、风险提示

1. **历史不代表未来**：任何基于历史数据的配置方案都无法保证未来收益
2. **再平衡纪律**：策略的有效性依赖于严格执行再平衡，切忌"涨了就不管""跌了就不卖"
3. **不要过度配置**：持有超过5-7只主动管理的股票型基金会产生"过度分散"问题，反而降低效率
4. **税务考虑**：基金持有期不足一年的分红可能涉及额外税负，合理规划赎回时点和金额
